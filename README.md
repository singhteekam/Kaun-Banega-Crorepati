# KBC

## A quiz app with features:
    - authentication using google
    - lifelines
    - show all results
    - show my results
    - random questions each time
    - timer for first 5 questions, etc.
    - Lifelines are: Audience Poll, Fifty-Fifty, Phone a friend, and Flip the question.
